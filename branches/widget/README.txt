ShareThis is a service that provides an unobtrusive way for your visitors to add your post to various social bookmarking sites, or send a link via e-mail to a friend.

== Installation ==

1. Download the plugin archive and expand it (you've likely already done this).
2. Put the 'sharethis.php' file into your wp-content/plugins/ directory.
3. Go to the Plugins page in your WordPress Administration area and click 'Activate' for ShareThis.
4. (Optional) Use the configurator to choose options for how to display your ShareThis window:

http://sharethis.com/publisher?type=wpplugin

then copy the code provided and paste it into the box in Options > ShareThis and click the update button.
